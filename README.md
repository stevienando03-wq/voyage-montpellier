# 🚆 Voyage Sud de France — circuit train sans voiture

Page web autonome qui présente le circuit **Paris → Montpellier → Nîmes → Arles → Avignon → Paris**
(fin juillet 2026, 2 voyageurs, 100 % train, logement gratuit à Montpellier).

👉 Fichier principal : [`index.html`](index.html) — double-clic pour l'ouvrir, aucune connexion requise.

## Publier sur GitHub Pages

1. Crée un dépôt sur GitHub (ex. `voyage-montpellier`), **public**.
2. Pousse ce dossier :
   ```powershell
   git remote add origin https://github.com/<ton-pseudo>/voyage-montpellier.git
   git branch -M main
   git push -u origin main
   ```
3. Sur GitHub → **Settings → Pages** → Source : `Deploy from a branch` → Branch : `main` / `/ (root)` → **Save**.
4. Au bout d'1–2 min, le site est en ligne sur :
   `https://<ton-pseudo>.github.io/voyage-montpellier/`

> Comme le fichier s'appelle `index.html`, GitHub Pages le sert automatiquement à la racine.
